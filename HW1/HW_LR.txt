
VuGen
	Записать скрипт для отправки сообщений через почту

	Нужно создать два почтовых ящика

	Первый сценарий скрипта:

	1. Авторизуемся в почте
	2. Зходим в ""Создать письмо""
	3. Пишем любой текст (текст должен быть уникальным)
	4. Отправляем письмо на второй почтовый ящик
	5. Использовать цикл, в котором будет 3 раза будет создано и отправлено письмо

	Второй сценарий:

	1. Авторизуемся на втором почтовом ящике
	2. Заходим в папку входящие письма
	3. Открываем рандомное письмо (не по порядку)
	4. Переносим письмо в папку прочитанные
	5. Если письма закончились, то вывести сообщение об этом

	В скрипте не должно быть файлов extrares (js, png, jpg и т.д.)
	Убрать из скрипта всё лишнее.

	Повторяющиеся хедеры убрать и сделать один общий с помощью функции web_add_auto_header

	В корреляции значения, которое будет использоваться при выборе письма
	использовать web_reg_save_param_regexp (регулярные выражения)

	В функции web_reg_save_param_regexp параметр Ordinal, должен принимать
	случайное значение. 
	Для параметра Ordinal написать функцию, которая будет возвращать случайное
	значение. Число в функции "рандом" должно быть равно количеству полученных параметров.

	Часть кода вынести в файл .h и .с.

	Активно пользоваться опцией help и stepToolBox

	Сохранить оба скрипта без лишних файлов и залить их в гит репозиторий.

Controller

	Создать 3 тест плана
	1. Поиск максимума
		Состоит из 7 ступеней
		Одна ступень длится 2 минуты
		На каждую ступень заходит по 5 пользователей
	2. Подтверждение максимума
		Подтверждение проходит на 4 ступени
	3. Тест надёжности
		70% от подтверждённого максимума

Analasys

	Сформировать отчёт
	В отчёте написать выводы по каждому тесту
	(Будем счиатать что в поиске максимума на 5 ступени началась деградация системы)
